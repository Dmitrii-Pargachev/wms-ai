# WMS·AI
Складская система для малого бизнеса, которая одновременно является сайтом для клиентов.
**Сайт:** https://wms-ai.ru (`{slug}.wms-ai.ru` - для клиентов и `{slug}.wms-ai.ru/system` - для продавца) 
## Описание
WMS-AI — платформа для тех, кто ведёт складской учёт и хочет автоматизировать процессы. Одна система закрывает сразу несколько задач: ведение склада, приёмку поставок, оформление продаж и сайт для клиентов с каталогом и заказами.
Каждый бизнес получает отдельный поддомен (например, myshop.wms-ai.ru), свою базу данных и полную изоляцию от других пользователей системы.
Платформа, объединяющая складской учёт и сайт для клиентов. Товары из склада автоматически появляются на публичном каталоге. Заказы с сайта списывают товары со склада.
## Функции
- Склад: товары, категории, поставки, продажи, автоматический пересчёт остатков
- Публичный сайт: каталог на поддомене `{slug}.wms-ai.ru` с корзиной и заказами
- Заказы с сайта с автосписанием товаров
- Загрузка и генерация ИИ фото товаров
- Журнал действий(хранение логов в системе в простом виде)
- Конструктор пользовательских таблиц (вести свои таблицы, если не хватает того, что предлагает сайт)
## Стек
- Backend: Django 4.2, Python
- База данных: SQLite (отдельная для каждого бизнеса)
- Frontend: Vanilla JS, инлайн CSS
- Сервер: Nginx → Gunicorn
- SSL: Let's Encrypt

## Либо зарегистрируйте аккаунт и слудйте инструкции:
Инструкция по работе
1. Регистрация и настройка сайта
 - Зарегистрировать аккаунт через почту. Перейти в Настройки → Сайт, заполнить данные (название, описание, телефон) и сохранить.
2. Склад
 - На странице Склад создать категорию товара и сам товар. Сделать поставку этого товара через раздел Поставки.
3. Фото товара
 - Перейти в ИИ генерация, создать карточку для товара с помощью нейросети и присвоить сгенерированную картинку товару.
4. Заказ с сайта
 - Перейти на сайт для клиента (ссылка на поддомен в настройках бизнеса), добавить товар в корзину и сделать заказ. Заказ отобразится во вкладке Заказы.
